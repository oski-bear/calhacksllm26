# Program Logo Sources

- `calfresh-food.svg`: CalFresh Food logo from the official CalFresh site, `https://calfresh.dss.ca.gov/food/logo.svg`.
- `california-wic.png`: California WIC logo from the official CDPH WIC media toolkit, `https://www.cdph.ca.gov/Programs/CFH/DWICSN/PublishingImages/Toolkit/WIC-Logo-WithTagline-RGB-EN%20(2).png`.
- `california-wic-trimmed.png`: Same WIC logo with only transparent canvas margins cropped for the card badge.
- `medi-cal-poppy.png`: Medi-Cal poppy logo image copied from Mono County's Medi-Cal page, `https://monocounty.ca.gov/sites/default/files/styles/full_node_primary/public/imageattachments/hhs/page/34711/medi-cal-tw-logo.png?itok=tglzfzqA`. The official DHCS Medi-Cal page identifies this as the "Medi-cal poppy logo": `https://www.dhcs.ca.gov/medi-cal/`.
- `liheap-energy.png`: Energy bill / thermostat icon from the official California Department of Community Services and Development LIHEAP page, `https://www.csd.ca.gov/PublishingImages/Pages/Home/Newest%20Thermostat%20v1.png`.
- `california-lifeline-icon.png`: Phone icon cropped from the publicly visible California LifeLine Program logo asset, `https://www.californialifelineprogram.com/ACPlogoFINAL.png`, as a quick hackathon visual placeholder.
- `calworks-logo.png`: CalWORKs logo from `https://www.calworks.org/img/calworks-logo.png`; the CDSS CalWORKs page links people to this CalWORKs campaign site.
- `hud-seal-white.png`: HUD seal from the official HUD site, `https://www.hud.gov/sites/default/files/Main/images/hud-seal-white4.png`, used for the Section 8 / Housing Choice Voucher card.

The UI keeps the source logo proportions intact and places each logo inside a circular badge.
