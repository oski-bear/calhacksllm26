# Program Logo Sources

- `calfresh-food.svg`: CalFresh Food logo from the official CalFresh site, `https://calfresh.dss.ca.gov/food/logo.svg`.
- `california-wic.png`: California WIC logo from the official CDPH WIC media toolkit, `https://www.cdph.ca.gov/Programs/CFH/DWICSN/PublishingImages/Toolkit/WIC-Logo-WithTagline-RGB-EN%20(2).png`.
- `california-wic-trimmed.png`: Same WIC logo with only transparent canvas margins cropped for the card badge.

The UI keeps the source logo proportions intact and places each logo inside a circular badge.
